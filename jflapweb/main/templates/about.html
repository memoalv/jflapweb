JFlapweb

What is “JFlapweb” exactly?

JFlapweb is an abstraction of the software called JFlap made for the web.

JFlapweb is the result of a social work project made for UACJ at the MATLAB. MATLAB is a project of one of my teachers at UACJ. It consists on doing integration of different tools and program for the research field made by students for the students. JFlapweb tries to integrate the java applet called Jflap in the context of the modern web standard. This will make the program more easy to use for the students, because now the software is not really required to be installed in their machines, so, this tool can be useful at the laboratory for doing research and it can be accesible for anyone at any time.
